# C Programming Project Suite

## Overview
This repository contains a collection of **console-based C programming mini projects** developed to practice and demonstrate core concepts of the C language.

The projects focus on structured programming, logical problem-solving, and real-world use cases suitable for **college assignments, mini projects, and viva examinations**.

## Projects Included
1. **Multifunction Calculator**
   - Performs basic arithmetic operations
   - Calculates area and perimeter of geometric shapes

2. **BMI Calculator**
   - Calculates Body Mass Index
   - Classifies results (Underweight, Normal, Overweight, Obese)

3. **Fruit Shopping Bill Generator**
   - Simulates shopping billing system
   - Applies discounts and GST

4. **Student Management System**
   - Uses structures and file handling
   - Add, search, display, and save student records

5. **Traffic Signal Emergency Handler**
   - Handles traffic logic with emergency conditions

6. **Bank Balance Calculator**
   - Uses different data types (int, float, double)
   - Calculates total balance

7. **Utility Programs**
   - Largest element in array
   - Factorial using recursion
   - Swap numbers using call by address
   - Print 2D array

## Concepts Covered
- Functions
- Arrays
- Structures
- File Handling
- Recursion
- Conditional Logic
- Data Types

## Tools Used
- C Programming Language
- GCC Compiler
- Console-based execution

## Learning Outcomes
- Improved understanding of C fundamentals
- Hands-on experience with real-world logic

## Conclusion
This project suite serves as a strong foundation for learning C programming and is suitable for **academic submissions and practice**.
